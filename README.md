## Controls
### Player movement:

- A: run to the left;

- D: run to the right

- W: jump

- S: Slide

### Debug keys:
- F1: first level start
- F2: start from the current level
- F5: save current state
- F6: Load the previous state
- F9: View Colliders
- F10: GodMode
- 1: Go to 1st Stage
- 2: Go to 2nd Stage
